Терминал Команды:

ls  - расшифровывает как "List files in the directory",
что переводится как "перечислить все файлы в папке" или "вывести на экран список файлов в папке".

cd <путь к папке> - расшифровывается как "Change directory",
что переводится как "сменить папку" или "перейти в другую папку".
Чтобы выйти обратно, напишем:
            cd ..

git init -
git branch - просмотр списков веток
git checkout - переключение между ветками
git log - просмотр истории коммита
git status - статус проверки файлов
git add и имя файла, который мы хотим отправить в GitHub - подготавливается к отправке
после того, как он подготовился к отправке, мы должны указать коммит, то есть короткое описание внесённые наши
изменения. Отвечает за это команда ...
git commit -m 'описание коммита в апострофах'

q - выход из терминала
